# yEdit 説明書

yEdit(ワイエディット)は、スクリーンリーダーに対応した、Windows 用のテキストエディタです。

この説明書では、yEdit の動作条件、使い方、制限事項、免責事項を説明します。

## 1. はじめに

yEdit は次のような特徴を持つテキストエディタです。

- **スクリーンリーダー対応**: NVDA など、UIA(UI オートメーション)に対応したスクリーンリーダーで読み上げできます。
- **タブ・検索置換・grep・自動バックアップ**: 日常のテキスト編集に必要な機能を一通り備えています。
- **マークダウンプレビューと CSV モード**: マークダウン文書の整形表示と、CSV ファイルを表として移動しながら読めるモードを備えています。

## 2. 動作条件

- **OS**: Windows 11(64ビット版)
- **ランタイム**: [.NET 9 デスクトップランタイム](https://dotnet.microsoft.com/ja-jp/download/dotnet/9.0)(x64)が必要です。未導入の場合はインストールしてください。
- **スクリーンリーダー**: NVDA で動作確認しています。UIA(UI オートメーション)に対応したスクリーンリーダーであれば読み上げできます。PC-Talker はサポート対象外です。

## 3. インストールと起動

yEdit はインストーラー不要です。配布されている zip ファイルを展開するだけで使えます。

1. 配布ページから `yEdit-(バージョン)-win-x64.zip` をダウンロードします。
2. zipファイルを解凍し、適当なフォルダにコピーします。
3. `yEdit.exe` を実行すると起動します。

アンインストールは、 yEdit のフォルダを削除するだけです。なお、設定とバックアップは別の場所(「[9. 設定](#9-設定)」「[10. バックアップと復元](#10-バックアップと復元)」参照)に保存されるため、完全に消したい場合はそれらのフォルダも削除してください。

## 4. 基本操作

### 4.1 画面構成

- **タイトルバー**: 「ファイル名 - yEdit」と表示されます。未保存の変更があるときは先頭に「* 」が付きます。
- **メニューバー**: ファイル(F)・編集(E)・検索(S)・読み上げ(R)・モード(M)・オプション(O)・ヘルプ(H)。Alt キーで移動できます。
- その他、いろいろ表示されています。

### 4.2 ファイルを開く・保存する

| 操作 | キー |
|---|---|
| 新規作成 | Ctrl+N |
| 開く | Ctrl+O |
| 上書き保存 | Ctrl+S |
| 名前を付けて保存 | メニューから |
| タブを閉じる | Ctrl+W |

対応する拡張子の初期値はテキスト(.txt)・マークダウン(.md)・CSV(.csv)の3つですが、「すべてのファイル」を選べば任意のテキストファイルを開けます。

### 4.3 文字コードと改行コード

ファイルを開くときは、文字コード(UTF-8 / Shift_JIS / EUC-JP / UTF-16 LE / UTF-16 BE)を自動判定します。判定を誤った場合(文字化けした場合)は、「ファイル」メニューの「文字コードを指定して開き直す」で正しい文字コードを選び直せます。

文字化けの可能性を検出した場合(変換できない文字があった場合)は、警告が表示されます。

新規作成時の文字コードと改行コード(CRLF / LF / CR)は、設定の「基本」タブで指定できます(既定は UTF-8・CRLF)。

### 4.4 タブ

複数のファイルをタブで切り替えながら編集できます。

| 操作 | キー |
|---|---|
| 次のタブ / 前のタブ | Ctrl+Tab / Ctrl+Shift+Tab |
| 1〜8番目のタブへ | Ctrl+1 〜 Ctrl+8 |
| 最後のタブへ | Ctrl+9 |
| タブを閉じる | Ctrl+W |

すでに開いているファイルをもう一度開こうとすると、新しいタブは作らずそのタブに切り替わります。最後の 1 つのタブを閉じると yEdit が終了します。

### 4.5 編集

元に戻す(Ctrl+Z)・やり直し(Ctrl+Y)・切り取り(Ctrl+X)・コピー(Ctrl+C)・貼り付け(Ctrl+V)・すべて選択(Ctrl+A)が使えます。

**挿入モードと上書きモード**: Insert キーで切り替わります。

**折り返し整形(禁則処理)**: Ctrl+Shift+J で、選択範囲(選択がなければ全文)を指定桁数で折り返し、実際の改行を挿入して整形します。行頭禁則・行末禁則・ぶら下げに対応しています。折り返し桁数は設定の「編集」タブ、禁則文字は「禁則処理」タブで変更できます。整形は 1 回の「元に戻す」で取り消せます。

なお、ファイルを書き換えずに見た目だけ折り返したい場合は、設定の「編集」タブにある「指定文字数で折り返す」を使ってください(保存内容は変わりません)。

### 4.6 最近使ったファイル

「ファイル」メニューの「最近のファイル」に、最近開いた・保存したファイルが最大 10 件表示されます。

## 5. 検索・置換・フォルダ検索(grep)

### 5.1 検索と置換

Ctrl+F で検索ダイアログ、Ctrl+H で置換ダイアログを開きます。ダイアログは開いたまま本文の編集に戻れます。

共通オプション:

- 大文字と小文字を区別する
- 単語単位で検索する
- 正規表現を使う
- 選択した範囲のみ(置換のみ)

検索語を入力すると一致件数が表示され、検索を実行すると「N 件中 M 件目」のように現在位置が通知されます。ダイアログを閉じたあとも、F3(次を検索)/ Shift+F3(前を検索)で検索を続けられます。ダイアログは Esc で閉じられます。

### 5.2 フォルダ検索(grep)

Ctrl+Shift+F で、フォルダ内の複数ファイルを横断検索できます。

- 検索対象のフォルダ、ファイル名のパターン(既定は `*.*`)、サブフォルダを含めるかどうかを指定できます。
- 検索中は進捗が表示され、Esc で中止できます。
- 結果は一覧ウィンドウに 1 件 1 行(ファイルの相対パス・行番号・行の内容)で表示されます。Enter で該当箇所を開いてジャンプします。
- 読み取れなかったファイルがあった場合は件数が通知されます。

## 6. 読み上げ機能

### 6.1 スクリーンリーダー対応

yEdit の本文と通知は UIA(UI オートメーション)を通じてスクリーンリーダーに提供されます。NVDA など UIA に対応したスクリーンリーダーの標準のテキスト読み上げがそのまま働き、特別な設定は不要です。

PC-Talker はサポート対象外です。

### 6.2 照会コマンド

編集中にいつでも現在の状態を確認できるコマンドです。

| 操作 | キー | 読み上げ内容 |
|---|---|---|
| 現在位置 | Ctrl+Alt+P | 「行 L / 全 N、桁 C、文字数 M」。選択中は選択文字数、上書きモード中は「上書き」も付きます |
| 行へ移動 | Ctrl+G | 行番号を入力してジャンプします(CSV モード中は「セルへ移動」になります) |

文字情報(カーソル位置の 1 文字の説明)は、お使いのスクリーンリーダー(NVDA/ナレーターなど)の標準機能をご利用ください。

## 7. マークダウンプレビュー

「モード」メニューの「マークダウンプレビュー」で、編集中の文書を整形表示します。

- 保存していない編集中の内容がそのまま反映されます。拡張子が .md でなくても使えます。
- 見出し・リスト・表・コードブロックなど、一般的なマークダウン記法(GFM 拡張含む)に対応しています。
- 文書と同じフォルダにある画像やファイルへの相対パスのリンクは、プレビュー内で表示・参照できます。
- プレビューは Esc キーまたは閉じるボタンで閉じて、編集に戻ります。

## 8. CSVモード

CSV ファイルを表(グリッド)として、セル単位で移動しながら読めるモードです。「モード」メニューの「CSVモード」でオン/オフを切り替えます。   
設定の「基本」タブで「.csv で自動 CSV モード」をオンにすると、.csv ファイルを開いたときに自動でこのモードになります。

CSV モード中は本文が読み取り専用になり、キー操作がセル移動に変わります。セルに移動すると、セルの内容に続いて行番号と列番号が読み上げられます。

### 8.1 CSVモードのキー操作

| キー | 機能 |
|---|---|
| 矢印キー(上下左右) | 隣のセルへ移動 |
| Tab | 現在のセルをもう一度読み上げ |
| C | 現在の列の見出し(最上段のセル)を読み上げ |
| R | 現在の行の見出し(左端のセル)を読み上げ |
| Home / End | 行の先頭 / 末尾のセルへ |
| PageUp / PageDown | 列の先頭 / 末尾のセルへ |
| Ctrl+Home / Ctrl+End | 一番左上 / 一番右下のセルへ |
| G または Ctrl+G | セルを指定して移動(「行,列」の形式で入力) |
| F2 | 現在のセルを編集 |

### 8.2 セルの編集(F2)

F2 を押すとセルの内容を編集する入力欄が開きます。

- **Enter**: 確定します(本文の CSV に反映されます)。
- **Alt+Enter**: セル内に改行を入れます。
- **Esc**: 編集を取り消します。

セル編集以外の方法で本文を書き換えたい場合は、CSV モードをオフにしてください。

## 9. 設定

「オプション」メニューの「設定」で設定ダイアログを開きます。5 つのタブに分かれており、OK を押すとすべての変更が保存され、即座に反映されます。

### 基本タブ

| 項目 | 内容 | 既定値 |
|---|---|---|
| 既定の文字コード | 新規作成時の文字コード(UTF-8 / Shift_JIS / EUC-JP / UTF-16 LE / UTF-16 BE) | UTF-8 |
| 既定の改行 | 新規作成時の改行コード(CRLF / LF / CR) | CRLF |
| .csv で自動CSVモード | .csv を開いたとき自動で CSV モードにする | オフ |
| 起動時に前回開いていたファイルを開く | 前回終了時に開いていたタブと未保存の変更を、次回起動時にそのまま復元する(終了時の保存確認は表示されなくなる。異常終了した場合も同じように復元される) | オフ |

### 編集タブ

| 項目 | 内容 | 既定値 |
|---|---|---|
| 指定文字数で折り返す | 表示上の折り返し(保存内容は変わりません) | オフ |
| 折り返し桁数 | 折り返しと整形の桁数(10〜1000) | 80 |
| タブ幅 | タブ 1 つの表示幅(1〜16) | 4 |
| タブをスペースに変換 | 新しく入力するタブをスペースにする | オフ |

### 禁則処理タブ

折り返し整形(Ctrl+Shift+J)で使う、行頭禁則文字・行末禁則文字・ぶら下げ文字を編集できます。

### 表示タブ

| 項目 | 内容 | 既定値 |
|---|---|---|
| フォント | 本文のフォントとサイズ | ＭＳ ゴシック 12pt |
| 配色 | 標準(白地に黒)/ 黒地に白 / 黒地に黄 / 黒地に緑 | 標準 |
| 行番号を表示する | 本文の左に行番号を表示 | オフ |
| 現在行を強調表示する | カーソルのある行に背景色を付ける | オフ |
| キャレットの太さ | カーソルの太さ(1〜5ピクセル) | 1 |
| 空白・改行文字を表示する | 空白やタブや改行を記号で可視化 | オフ |

### バックアップタブ

| 項目 | 内容 | 既定値 |
|---|---|---|
| 文書のバックアップを有効にする | 自動バックアップのオン/オフ | オン |
| バックアップ間隔(秒) | 5〜3600 秒 | 300 |
| 起動時に復元するか確認する | 復元前に確認ダイアログを出す | オン |

### 設定の保存場所

設定は `%APPDATA%\yEdit\settings.json` に保存されます。

## 10. バックアップと復元

yEdit は、保存していない変更を一定間隔(既定 300 秒)で自動的にバックアップします。バックアップは `%APPDATA%\yEdit\backups` に保存され、正常に終了したときに自動で削除されます。

強制終了や電源断などで異常終了した場合、次回起動時にバックアップが見つかると復元の確認ダイアログが表示され、次の操作を選べます。

- **復元する**: 選択した文書を復元します。
- **すべて破棄**: バックアップを削除して通常どおり起動します。
- **あとで**: バックアップを残したまま起動します(次回起動時にまた確認されます)。

設定で「起動時に復元するか確認する」をオフにすると、確認なしで自動的に全件復元されます。

「起動時に前回開いていたファイルを開く」が有効な場合は、異常終了時も含めて前回の状態(タブ構成と未保存の変更)が自動的に復元されるため、上記の復元の確認画面は表示されません(「起動時に復元するか確認する」はこの設定が無効のときにのみ働きます)。

「起動時に前回開いていたファイルを開く」を有効にしている場合、終了時に未保存の変更があっても保存の確認は表示されず、次回起動時に編集途中の状態のまま復元されます。ただし変更内容が極端に大きい場合(1 ファイルあたりおよそ 3200 万文字を超える場合)や、文書のバックアップを無効にしている場合は、通常どおり保存の確認が表示されます。

バックアップはあくまで異常終了への備えです。大切な文書はこまめに Ctrl+S で保存してください。

## 12. 制限事項・免責事項

### 12.1 制限事項

- **マークダウンプレビューのサイズ上限**: 内部の仕組み上、約 2MB を超える文書はプレビューできないことがあります。
- **マークダウンプレビューでの外部画像**: 安全のため、インターネット上の画像(外部 URL の画像)はプレビューに表示されません。文書と同じフォルダなど、ローカルにある画像は表示されます。
- **CSV モード中の編集**: CSV モード中は本文が読み取り専用になり、セル編集(F2)以外の書き換え・置換・折り返し整形はできません。
- **複数同時起動**: yEdit を複数同時に起動することはできますが、設定とバックアップの保存場所が共有されるため、バックアップの復元動作などが正しく働かない場合があります。複数起動での利用は保証外です。
- **文字コードの自動判定**: 自動判定は完全ではありません。文字化けした場合は「文字コードを指定して開き直す」で選び直してください。
- **PC-Talker**: PC-Talker はサポート対象外です。読み上げには NVDA など UIA(UI オートメーション)対応のスクリーンリーダーをご利用ください。
