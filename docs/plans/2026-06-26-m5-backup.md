# M5 自動バックアップ＆クラッシュ復元 — 実装計画

**Goal:** 定期バックアップ（サイドカー・背景直列書込・無音）と起動時の復元提案。Core は SR/スレッド非依存で単体テスト可能。

**Architecture:** `yEdit.Core.Backup`（`BackupRecord`/`BackupStore`＝原子的 JSON サイドカー）＋ `yEdit.App` の `BackupCoordinator`（Timer＋Reconcile＋背景直列ライター＋掃除）/`RestoreDialog`（CheckedListBox）。設計根拠: `docs/plans/2026-06-26-m5-backup-design.md`。

**Tech Stack:** .NET 9 / C# / WinForms / `System.Text.Json` / xUnit。

## 前提・規約
